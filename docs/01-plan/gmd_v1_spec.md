## 1. 목적

**이수현 본부장·대표이사·영업본부가 매주 월요일 09:00에 한 장을 보고 30초 안에 "지난주/이번달 무엇이 달라졌는가"를 파악하게 한다.**

현재 GMD 대시보드는 매출 현황을 보여주지만, "무엇을 해야 하는지"에 대한 액션 신호가 약합니다. v1은 아래 3가지를 추가합니다.

1. **집중도·리스크 신호** — 매출이 한 곳/한 사람에 쏠려 있는 현실을 숨기지 않는다
2. **이탈 위험 거래처 조기 경보** — 숫자가 터지기 전에 경영진이 먼저 안다

## 3. KPI 7개 상세 스펙

각 KPI는 `(name, 계산식, 시각화, 갱신주기, 경보임계)` 순으로 정의합니다.

### KPI-1. 월별 매출 추이 (추정)
- **계산식**: `PRT_ESTH.AM_SUPPLY` 월별 합계, 최근 13개월
- **시각화**: 라인차트 + 3개월 이동평균선 (**이동평균은 완료월만으로 계산, 당월은 점으로만 표시**)
- **당월 표시**: "4월 MTD: __억 (경과 __일)" 텍스트로 명기 — 예상 매출은 **v1에서 제외** (월말 세금계산서 발행 이후에야 확정 수치 가능. ERP 개편 후 v2)
- **갱신**: 일 1회 (daily_update.sh 기존 파이프)
- **경보**: 직전 완료월 vs 3개월 이동평균 -20% 이하면 빨강 (**당월은 경보 대상 제외** — 월 초 오발동 방지)
- **검증값(2025)**: 월 17~31억, 연 270억

### KPI-2. 거래처 매출 Top 10 + 집중도 경고
- **계산식**: 연누계 `PRT_ESTH.AM_SUPPLY` 거래처별 내림차순 Top 10, HHI(허핀달지수) 동시 계산
- **시각화**: 파레토 차트 (막대+누적선), 상단에 "Top 10 집중도: __%" 큰 숫자
- **경보**: Top 1 비중 ≥ 20% 또는 Top 5 비중 ≥ 60% → 주황 경고 라벨
- **검증값(2025)**: Top 1 교원구몬 22%, Top 5 54%, Top 10 70% → **경보 조건 충족 중**

### KPI-3. 영업자별 실적 + 1인 리스크
- **계산식**: `PRT_ESTH.CD_EMP`별 금액·건수·담당거래처수
- **시각화**: 막대차트 Top 15 + 상단에 "1인 최대 매출 비중: __%"
- **경보**: 한 영업자의 매출 비중 ≥ 25% → 주황 경고 ("키맨 리스크")
- **검증값(2025)**: 방진혁 30% → **즉시 경보 발동**
- **액션 연계**: 경보 클릭 시 "이 영업자 담당 Top 3 거래처 이탈 시 영향액" 팝업

### KPI-4. 신규 vs 기존 거래처 매출 믹스
- **계산식**: 당년 견적 거래처 중 "과거 3년(2022~전년) 거래 이력 없음" = 신규
- **시각화**: 도넛차트(금액 기준) + 아래 표(거래처 수, 평균 건단가)
- **갱신**: 일 1회
- **검증값(2025)**: 신규 79곳·24억(9%) / 기존 211곳·246억(91%)

### KPI-5. 이탈 위험 거래처 (조기 경보 패널)
- **계산식**:
  - 직전 2년 누적 견적 ≥ 5천만원 거래처 중
  - 최근 **90일 이상** 신규 견적 0건
  - 마지막 견적일 역순 정렬
- **시각화**: 빨강 경고 배지 + 거래처 리스트 (누적매출 / 마지막견적일 / 경과일수 / 담당영업자)
- **갱신**: 일 1회
- **액션**: 담당 영업자 이름·전화번호 같이 표시 → 클릭 시 이수현 본부장에게 "이 거래처 왜 조용한가" 질문 보낼 수 있는 Slack 연동 (v1.1)
- **검증값(2026-04-15 기준)**: 씨디유파트너스 26억·167일, GS리테일 19.5억·328일, 새마을금고복지회 8.8억·210일

### KPI-6. 월별 용지 매입액 (원가 선행 지표)
- **계산식**: `viewGabwoo_마감` 월별 `공급가액` 합, 제조사(무림/한솔/기타) 분리
- **시각화**: 스택 막대 + 매출 대비 비율 % 보조선
- **경보**: 용지비/매출 비중이 3개월 평균 대비 +5%p 상승 시 주황 (수익성 압박 신호)
- **검증값(2025)**: 월 4~8.8억, 연 64억, 매출 대비 평균 23%

### KPI-7. 품목 카테고리 믹스 (Top 20)
- **계산식**: `PRT_ESTL.UM × QT` 품목별 합, 연누계 (라인 AM_SUPPLY는 0이므로 사용 불가)
- **NULL/빈 CD_ITEM 처리**: "(미분류)" 단일 버킷으로 묶음
- **시각화**: 트리맵 + 품목 클릭 시 "해당 품목 매출 상위 거래처 Top 5" 드릴다운
- **갱신**: 일 1회
- **검증값(2025)**: K편E골(3.9억), S04(8.3억), UV판(8억), PP필름·백상지 등

## 6. v1에서 **의도적으로 뺀 것**

| 항목 | 이유 |
|---|---|
| 견적→수주 전환율 | ERP 입력 방식상 99% 고정 → 지표 의미 없음 (실제로는 수주 뒤 사후 입력하는 역순 프로세스) |
| 수주→작업지시 리드타임 | 해석 기준(견적→WO vs 수주→WO) 사전 논의 필요 → v1.5+ |
| 거래처별 실마진 | 원가 매칭 미검증, v1.5 베타로 분리 |
| 설비 다운타임 | 컬럼명 재확인 필요, v1.1 |
| 클레임/반품 | `PRT_CLAIMH` 0건, 종이 기록 의심 |
| 예산 대비 실적 | 계획 테이블 `SAL_CUSTPLANMM` 데이터 미확인 |
| 패키지사업부 분리 | 데이터 위치 미확인 (이본부장/G2SYS 확인 필요) |

## 7. v1.5 예고 — 교원구몬 실마진 파일럿 (베타)

### 범위
- 대상: 교원구몬 1곳 (2025년 견적 59.8억, 1,345건)
- 목표: 건별로 `매출(견적) − 용지원가 − 외주원가`를 계산하여 **매출총이익 근사치** 산출
- **라벨링**: "실험용, 숫자 확정 아님" 배지 상시 노출. 경영진 전체 대시보드에 올리지 않고 **이수현 본부장·교원구몬 담당 이사 2명에게만** 베타 접근 제공

### 선행 검증 (오늘 저녁 18:00 이후 실행)
1. `PRT_WO.NO_WO` 포맷 샘플 확인
2. `SCT_POL.NO_WO` ↔ `PRT_WO.NO_WO` 직접 조인 매칭율
3. `PUR_POL.NO_WO` 매칭율 (외주비 포함용)

### 검증 통과 기준
- 교원구몬 2025 수주건 중 용지원가 매칭율 **85% 이상**일 때만 베타 공개
- 미만이면 원가 구조 재설계 → v2.0으로 이관

### 본질적 한계 (대시보드에 명시)
- 인쇄 공정비·인건비·설비 감가상각·간접비 미포함
- 따라서 이 수치는 **"매출총이익(GP) 근사치"**이지 순이익이 아님
- 교원구몬 담당 이사와 월 1회 숫자 대조 미팅 정례화 (라포 활용)

## 9. 리스크 & 완화

1. **PRT_ESTH가 진짜 매출이 맞는지 이수현 본부장 구두 확인 필요**
   - 견적액과 실제 세금계산서 금액 괴리가 평균 몇 %인지 모름
   - 완화: 4/21 검토 미팅 때 교원구몬 2025 세금계산서 총액 1건 대조
2. **KPI-5 이탈 위험 경고를 보고 영업자가 방어적으로 반응할 가능성**
   - 완화: 이본부장이 먼저 보고 → 일방 통보 아닌 지원 관점으로 영업자에게 공유
3. **패키지사업부 데이터 미포함 → 사장님 기대 어긋날 수 있음**
   - 완화: v1 상단에 "이 화면은 출판사업부 기준" 명시, 패키지는 v2 예고
4. **매일 갱신이 ERP 운영 서버에 부담 가능**
   - 완화: 기존 daily_update.sh가 이미 07:00에 돌고 있음. 같은 배치에 붙여서 추가 연결 0회
5. **Jack Mac이 배치 서버인 SPOF** (CTO 리뷰 지적)
   - v1 완화: Slack 알림으로 실패 즉시 감지 가능
   - v2 이관: 사내 상시 PC로 배치 호스트 이전 (Go-Live 후 1개월 내)
